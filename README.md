# 🪣 Sandbox - Mon Espace d'Expérimentation

Bienvenue dans mon "bac à sable" ! 

Ce repository est mon espace de brouillon et d'entraînement personnel. J'y regroupe tous mes tests de code, mes petits scripts, mes exercices de cours et mes expérimentations dans différents langages de programmation.

## 📂 Structure du Repository

Pour garder cet espace un minimum organisé, les codes sont triés par langages ou par thématiques. Voici la structure actuelle :

* **`/projets-brouillons`** : Tests rapides ou codes cassés en cours de réparation (bientôt de vrais projets !).
* **`/c`** : Apprentissage des bases de la mémoire et algorithmique bas niveau.
* **`/cpp`** : Programmation orientée objet et concepts avancés en C++.
* **`/python`** : Scripts, tests et exercices d'algorithmique.
* **`/powershell`** : Scripts d'automatisation et d'administration système Windows.
* **`/java`** : Programmation orientée objet et création d'applications.
* **`/php`** : Développement web côté serveur et logique backend.
* **`/mysql`** : Création de bases de données et entraînement aux requêtes SQL.

*(Note : Cette structure évolue au fil de mon apprentissage)*

## 🚀 Pourquoi ce repository ?

* **Centraliser** mon apprentissage au même endroit.
* **Garder une trace** de ma progression (mes erreurs comme mes réussites).
* **Créer une boîte à outils** avec un accès rapide à des bouts de code réutilisables pour le futur.
